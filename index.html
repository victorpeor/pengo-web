Recreation of the 1982 arcade classic

Game Description

Este proyecto es una reinterpretación del juego arcade Pengo (Sega, 1982), donde el jugador empuja bloques de hielo para derrotar enemigos y sobrevivir. El juego ha sido desarrollado en C++ usando la librería Raylib como proyecto académico de la UPC.

Team

Víctor Pérez Ortega: implementation of sound, enemies, map generation,e tc
Mario Torrents Rodríguez: coding, design
